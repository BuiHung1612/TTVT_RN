Tối ưu Hiệu năng

Hiệu năng là yếu tố then chốt quyết định trải nghiệm người dùng. Dưới đây là các kỹ thuật tối ưu hóa phổ biến:

Tránh re-render không cần thiết

  • Sử dụng React.memo
  • Sử dụng useCallback và useMemo
  • Tối ưu useEffect dependencies

Tối ưu hiển thị danh sách

  • Sử dụng FlatList thay vì ScrollView
  • Implement getItemLayout
  • Sử dụng keyExtractor hiệu quả

Tối ưu hình ảnh

  • Resize ảnh phù hợp với resizeMode
  • Sử dụng react-native-turbo-image
  • Lazy loading với onLoad

Tối ưu Bundle size

  • Sử dụng Flipper để phân tích bundle
  • Tree shaking và import chọn lọc
  • Code splitting với React.lazy
