Mô hình Kim tự tháp Kiểm thử

Phân bổ các loại kiểm thử khác nhau theo tỷ lệ thích hợp

Unit Tests (Jest + RTL)

Jest cho logic, hooks và utils. React Native Testing Library cho component testing. Kiểm tra functions, custom hooks và component behaviors cô lập.

Component Tests (RTL)

Sử dụng React Native Testing Library để test UI components. Kiểm tra rendering, user interactions, props và state changes một cách isolated.

Integration Tests (Detox)

Detox framework để test tương tác giữa các screens, navigation, API calls và complex user flows. Chạy trên simulator/emulator thực tế.

End-to-End Tests (Appium/Maestro)

Appium hoặc Maestro cho E2E testing trên device thật. Test complete user journeys, critical business flows và performance trên production builds.
