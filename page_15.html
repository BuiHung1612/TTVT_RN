Gỡ lỗi (Debugging)

Gỡ lỗi là một phần không thể thiếu trong quá trình phát triển ứng dụng. Các công cụ hiện đại giúp việc này trở nên dễ dàng và hiệu quả hơn.

Flipper

  • Một nền tảng gỡ lỗi đa năng cho ứng dụng di động
  • Kiểm tra Network: Theo dõi yêu cầu HTTP
  • Kiểm tra Layout: Hiển thị cấu trúc UI
  • Kiểm tra Performance: Theo dõi CPU, bộ nhớ

React DevTools

  • Một extension trình duyệt cho kiểm tra React
  • Kiểm tra cây thành phần React
  • Xem và chỉnh sửa trạng thái (state) và props
  • Hỗ trợ việc theo dõi luồng hoạt động

Reactotron

  • Inspect state, actions, API calls, and logs in real-time
  • Great for tracking Redux/Rematch or custom state flows
  • Simple to add and very low overhead during development

Gỡ lỗi trực tiếp

  • React Native cho phép gỡ lỗi trực tiếp trong Chrome
  • Thông qua extension trong VS Code
  • Cung cấp các tính năng breakpoint
  • Xem biến và thực thi mã từng bước
Gợi ý: Dùng Reactotron khi bạn muốn debug state / actions / HTTP request trực tiếp trong môi trường phát triển. Dùng Flipper nếu cần inspect layout, network và performance cùng lúc.
