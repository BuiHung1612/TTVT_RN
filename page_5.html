Cơ chế hoạt động của React Native

React Native chuyển mã JavaScript thành UI native qua chuỗi bước rõ ràng: Runtime → Renderer → Giao tiếp native → Quản lý UI → Render hệ điều hành.

JS Runtime

Chạy React (JSX, state, logic) trên Hermes/JSC.

Fabric Renderer

Tạo Shadow Tree và tính layout bằng Yoga.

JSI & Native Call

Gọi API native trực tiếp (không cần JSON Bridge).

Native UI Manager

Nhận lệnh mount/update, thao tác view native.

Render UI Native

Hệ điều hành hiển thị UI (iOS/Android).

React suy nghĩ Fabric sắp xếp Native vẽ lên màn hình
