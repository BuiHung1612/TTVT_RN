Ví dụ về Zustand

Zustand là một giải pháp quản lý trạng thái nhẹ nhàng, hiện đại cho React Native. Dưới đây là ví dụ cơ bản về cách sử dụng Zustand để quản lý trạng thái đếm:

useCounterStore.js
1 // Import Zustand
2 import create from 'zustand';
3
4 // Định nghĩa store
5 const useCounterStore = create(set => ({
6 count: 0,
7 increment: () => set(state => ({ count: state.count + 1 })),
8 decrement: () => set(state => ({ count: state.count - 1 })),
9 }));
10
11 // Sử dụng trong component
12 function Counter() {
13 const { count, increment, decrement } = useCounterStore();
14
15 return (
16 <View>
17 <Text>Count: {count}</Text>
18 <Button title="Tăng" onPress={increment} />
19 <Button title="Giảm" onPress={decrement} />
20 </View>
21 );
22 }

Giải thích

  • create(set => ({...})): Hàm tạo store, nhận vào một factory function
  • set(state => ({ count: state.count + 1 })): Cập nhật trạng thái dựa trên trạng thái trước
  • useCounterStore(): Hook để truy cập store

Ưu điểm

  • Đơn giản, dễ học và dễ sử dụng
  • Hiệu năng tốt, ít re-render không cần thiết
  • Không cần boilerplate như Redux
